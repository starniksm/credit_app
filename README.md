# Credit Application System

Это веб-приложение для обработки заявок на кредит в кредитном отделе. Приложение позволяет принимать заявки, оценивать кредитоспособность клиентов и управлять процессом одобрения кредитов.

## Особенности

- Веб-интерфейс для приема заявок на кредит с валидацией данных
- Система хранения заявок в базе данных PostgreSQL с возможностью фильтрации и поиска
- Интеграция с внешним API для получения данных из бюро кредитных историй (симуляция через mock-сервер)
- ИИ-модуль для анализа кредитной истории и оценки платежеспособности
- Система уведомлений и логирования действий
- Административная панель для просмотра статуса заявок
- Возможность экспорта отчетов в форматах PDF/CSV/JSON

## Требования

- Go 1.21+
- PostgreSQL
- Docker (по желанию)

## Установка и запуск

### Локальный запуск

1. Клонируйте репозиторий:
```bash
git clone <repository-url>
cd credit_app
```

2. Установите зависимости:
```bash
go mod tidy
```

3. Установите переменные окружения для подключения к базе данных:
```bash
export DB_HOST=localhost
export DB_PORT=5432
export DB_USER=postgres
export DB_PASSWORD=postgres
export DB_NAME=credit_db
```

4. Запустите приложение:
```bash
go run main.go
```

Приложение будет доступно по адресу http://localhost:8080

### Запуск с помощью Docker

1. Убедитесь, что у вас установлен Docker и Docker Compose

2. Запустите приложение:
```bash
docker-compose up --build
```

Приложение будет доступно по адресу http://localhost:8080, а база данных будет доступна на порту 5432.

## API Endpoints

### Публичные эндпоинты

- `POST /api/auth/login` - Авторизация пользователя
- `POST /api/auth/register` - Регистрация нового пользователя
- `GET /health` - Проверка состояния сервера

### Защищенные эндпоинты (требуют авторизации)

#### Управление заявками

- `POST /api/applications` - Создать новую заявку на кредит
- `GET /api/applications` - Получить список заявок (с фильтрацией)
- `GET /api/applications/:id` - Получить информацию о конкретной заявке
- `PUT /api/applications/:id` - Обновить заявку
- `DELETE /api/applications/:id` - Удалить заявку

#### Административные эндпоинты

- `GET /api/admin/dashboard/stats` - Получить статистику для дашборда
- `PUT /api/admin/applications/:id/status` - Обновить статус заявки
- `GET /api/admin/applications/export?type={format}` - Экспорт заявок (CSV/PDF/JSON)
- `GET /api/admin/applications/review` - Получить заявки на ручную проверку
